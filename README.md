## Pokedex

Bu proje, Pokémon bilgilerini listeleyen Pokedex uygulamasıdır. React kullanılarak geliştirilmiştir.

## Özellikler

- Pokémon listesi
- Pokémon detayları (tip, yetenekler, istatistikler vb.)

## Kullanılan Teknolojiler

- React
- API
- React Router 
- CSS
